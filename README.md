## File Sorter in Java

Program I use to automate image sorting.
Example script usage below.

Folder source:
```
source/
    image1.png (created 05.2022)
    image2.jpg (created 05.2022)
    image3.jpg (created 02.2022)
```

Result in folder desination:
```
destination/2022/
    2/
      image3.jpg 
    5/
      image1.png 
      image2.jpg 
```

Program arguments:
- source path
- destination path
